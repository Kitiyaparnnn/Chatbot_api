## Description

This repository is a server part of Chatbot for Education Service. In this part, it is used to calculate complex conditions such as characteristic conversion and expulsion rules.
API for this server is https://chatbotapi-production.up.railway.app/

## Git Clone

```bash
$ git clone 
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
